
Body Mass Index(BMI) calculater application in take a user data Age,
Gender, Height(cm) and Weight(kg) with a user-friendly interface. 

Classification BMI range - kg/m2
Severe Thinness < 16
Moderate Thinness 16 - 17
Mild Thinness 17 - 18.5
Normal 18.5 - 25
Overweight 25 - 30
Obese Class I 30 - 35
Obese Class II 35 - 40
Obese Class III > 40

Docker Commad:-

docker pull nginx:latest
docker run -d -p 8080:80 --name my-web-server nginx
docker stop my-web-server
